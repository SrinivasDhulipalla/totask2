Welcome to totask2.

a simple show case for some common application libraries.

SpringMVC web application in Java 8 using some closures https://java.com/de/[java]. Templating Engine is thymeleaf.

client side http://getbootstrap.com//[bootstrap^] css, http://bootstrapvalidator.com/[bootstrapvalidator^] for client side validations, http://jquery.com/[jquery^] as JavaScript library, adding http://www.datatables.net/[datatables^] as datagrid. Charts are done with http://www.chartjs.org/[Charts.js^]. Autocompletion with REST/json using http://jqueryui.com/autocomplete/[autocomplete plugin^].

Basic reporting with http://community.jaspersoft.com/project/jasperreports-library[JasperReports^].

peristance done on http://www.h2database.com/[h2^], migrations (DDLs) done by http://flywaydb.org/[flyway^]. documentation with http://asciidoctor.org/docs/asciidoc-writers-guide/[asciidoc^] markup.

Build management by http://www.gradle.org/[gradle^], adding some plugins like checkstyle, pmd, findbugs, asciidoctor, flyway
